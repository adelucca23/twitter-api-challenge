<h1>Twitter API Code Challenge</h1>

## Requirements
- [Node](https://nodejs.org/en/download/) and NPM are installed and up-to-date


## Getting Started
1. **Clone the repository.**
1. **Install project dependencies.**
    
    ```
    $ npm i
    ```
    
1. **Start the development server.**
    
    ```
    $ npm run dev
    ```
